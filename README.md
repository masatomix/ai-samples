# ai-samples


GooラボのAPI実行サンプル
```
# cd samples
# sudo pip3 install -r requirement.txt -t ./
# sudo python3 goolab.py
```



GooラボのAPIから固有表現を取得して、教師データを作成するサンプル。
教師データの元ファイル500から、0〜30を使用する場合:
```
# cd samples
# sudo pip3 install -r requirement.txt -t ./
# sudo python3 create_teach_data.py 0 30
```
